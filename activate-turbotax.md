# TurboTax Activation Guide with License Code

Activating TurboTax is required to use all features.

## Steps to Activate

1. Open TurboTax software  
2. Click on “Activate”  
3. Enter your license code  
4. Click submit  

## Where to Find License Code

- Email after purchase  
- Product box  
- Retail receipt  

## Activation Issues

- Invalid code  
- Server error  

## Fix

- Recheck code  
- Connect to internet  
- Contact support
