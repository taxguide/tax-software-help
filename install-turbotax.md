# How to Install TurboTax on Windows and Mac

After downloading, follow these steps to install TurboTax:

## Installation Steps

1. Open the downloaded file  
2. Double-click the setup file  
3. Click “Install”  
4. Follow on-screen instructions  
5. Wait for installation to complete  

## Tips

- Close all background apps  
- Ensure enough disk space  
- Run as administrator  

## Common Errors

- Installation stuck  
- Setup not opening  

## Fix

- Restart system  
- Re-download file
